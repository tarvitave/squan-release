# Squan Release Changelog

## v2.0.4 (2026-04-06)

### New Features
- ✅ Added Recipes view with 6 workflows (Code Review, Bug Fix, Refactor, Tests, Docs, Feature)
- ✅ Added Extensions view with 4 MCP tools (Filesystem, GitHub, Web Search, Database)
- ✅ Fixed server connection - "Connect to Server" now opens browser automatically
- ✅ Enhanced navigation with 5 sections (Chat, Recipes, Extensions, Settings, Server)

### Improvements
- Professional UI with gradient design
- Clear messaging about Server Mode features
- Better user onboarding
- Settings persistence working

### Bug Fixes
- Fixed `open_external` command for opening URLs in browser
- All Tauri v2 compatibility issues resolved
